ASCII 网点 · 关键词标注 · 氛围动效
⌁

拖入或上传一张照片开始，或随手来一张

浏览器内识别画面物体，映射成氛围词。识别后所有标签可拖动、改字、删除。

高光网点：只在画面亮部撒字符（像 pixpic 那样顺着受光区域长出来），整图生效、不用框。想只处理局部时可切「框选遮罩」。

叠加动效图层，可多选。每个动效能单独框定范围（像图里只有灯下 / 伞下有动效），不框=全屏。导出 WebM 会录下动画。

挑一个形状 + 一种动效，组合成一层。「自动」会按画面内容联想形状（云朵/冰淇淋…），也能贴自定义 emoji。

手绘装饰

手绘描边装饰，会自己描出来 + 持续微动，导出 WebM 也录得到。先选颜色，再点元素。

提示：点画面上的装饰可选中，再拖动 / 改大小 / 删除。

像素故障 / 千禧梦核风。四种故障元素可叠加，自动跳位 + 闪烁，导出视频/GIF 录得到动态。每种都能单独框范围（去「动效」tab 设）。

图生网格渐变（参考 photogradient）：把照片的颜色采样成柔滑的弥散渐变，颜色与位置都来自画面本身。混合越高越抽象，网格越小越梦幻。

噪点 · 颗粒

叠一层胶片颗粒 / 噪点质感，复古朦胧更耐看。强度 0 = 关闭。

横版照片更适合：把画面切成两半，一半是原图，一半是从图里提取的主色调色板。ASCII / 动效 / 文字都跨越整张图，自然把两半串在一起。

PNG = 当前画面静图；视频/GIF = 录制带动效的动画。GIF 到处自动播但 256 色、文件较大；视频更清晰（优先 MP4）。
